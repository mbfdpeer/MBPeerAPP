# Miami Beach Fire Department — CISM Support Page

A mobile-optimized, single-file HTML resource page for the **Miami Beach Fire Department Critical Incident Stress Management (CISM)** peer support program.

## Purpose

Provides firefighters and rescue personnel with a fast, confidential, one-tap path to peer support following traumatic or difficult incidents.

## Features

- Fully mobile-optimized (max-width 480 px, responsive down to 320 px)
- Flat design — no gradients, all right-angle corners
- Sticky header and sticky bottom call-to-action bar
- Color-coded card system (blue / red / gold / green) for quick visual scanning
- MBFD department seal in the footer with issue date and authorization info
- Zero external dependencies beyond Google Fonts (Montserrat)

## Usage

1. Open `index.html` in any modern browser — no build step required.
2. Replace all `[INSERT …]` placeholders with real contact names and phone numbers before publishing.
3. Keep `MBFD.jpg` in the same directory as `index.html` so the footer logo loads correctly.

## Placeholders to fill

| Placeholder | Description |
|---|---|
| `[INSERT CISM COORDINATOR NAME]` | Name of the primary CISM coordinator |
| `[INSERT CISM PHONE NUMBER]` | Direct phone number for CISM contact |
| `[INSERT-CISM-PHONE]` | Phone number used in `tel:` href links |
| `[INSERT BACKUP CONTACT]` | Secondary contact name/number |
| `[INSERT PEER CHAPLAIN CONTACT]` | Chaplain name/number |
| `[INSERT OPTIONAL EMAIL / INTERNAL RESOURCE LINK]` | Optional email or intranet link |
